Languages

Standard Languages

  • Common: The lingua franca of Maerund, this language is spoken by most civilized races of Maerund.
  • Vrillian: This language is spoken by the Vrillian Human ethnic group.
  • Serelic: This language is spoken by the Serelic Human ethnic group.
  • Tyrili: This language is spoken by the Tyrili Human ethnic group.
  • Eldaarish: This language is spoken by Eldaari.
  • Feldani: This language is spoken by Feldani.
  • Runic: This language is spoken by Dwerger.
  • Ul'Raathan: This language is spoken by Ul'Raathi.
  • Saurian: This language is spoken by Saurians.
  • Magnar: This language is spoken by Magnians.
  • Daitien: The official language of the Dragon Empire, this language is spoken by Tatsu, Tengu, Genbu, Tesso, and Kitsune.
  • Sign Language: This is not a single language, but a method of visual communication using hand signals. One can only learn sign language in a language they already know, and they must learn sign language again for each additional language they wish to be able to use sign language for.

Exotic Languages

  • Abyssal: This language is spoken by chaotic evil outsiders, such as demons.
  • Celestial: This language is spoken by good outsiders, such as angels.
  • Deep Speech: This language is spoken by aberrations, such as mind flayers and aboleth.
  • Draconic: This language is spoken by dragons and related races, such as true dragons and kobolds.
  • Giant: This language is spoken by giants, such as ogres and trolls.
  • Gnoll: This language is spoken by gnolls.
  • Goblin: This language is spoken by goblinoids, such as goblins and bugbears.
  • Infernal: This language is spoken by lawful evil outsiders, such as devils.
  • Orcish: This language is spoken by orcs.
  • Primordial: This language is spoken by elementals, such as efreet and marids. Auran, Aquan, Ignan, and Terran are dialects from the four elemental planes that are all similar enough to be used for communication with speakers of each dialect.
  • Sylvan: This language is spoken by fey, such as nymphs and dryads.
  • Undercommon: The lingua franca of the Under Realms, this language is spoken by most subterranean races.
